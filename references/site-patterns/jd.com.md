## 平台特征

- **架构**: SPA单页应用，商品列表通过AJAX动态加载
- **加载方式**: 懒加载机制，页面底部显示"正在奋力加载中"
- **反爬**: 轻度，CDP模式正常访问
- **登录需求**: 无需登录即可查看商品和价格

## 有效模式

### 搜索URL格式
```
https://search.jd.com/Search?keyword={搜索词}
```

### 商品列表选择器
```javascript
// 主要选择器
#J_goodsList .gl-item

// 备选选择器
.goods-list-v2 li
[data-sku]
```

### 商品信息提取
| 字段 | 选择器 |
|------|--------|
| 标题 | `.p-name em` 或 `.p-name a` |
| 价格 | `.p-price i` 或 `.J_price i` |
| 店铺 | `.p-shop a` |
| 链接 | `.p-name a` |

## 已知陷阱

1. **页面加载延迟**: 首次加载时部分商品区域显示灰色占位块，需要等待或滚动触发懒加载
2. **选择器变化**: 京东可能更新页面结构，多种选择器备用
3. **网络监控**: 商品数据通过内部API加载，但URL不固定，建议直接DOM提取

## 操作建议

1. 打开搜索页后等待3-5秒让商品列表加载
2. 使用滚动触发更多商品加载
3. 提取数据时优先使用 `#J_goodsList .gl-item` 选择器
